# AuxSync

AuxSync is a multiplayer party game where players compete to find the **song that best syncs with a GIF**. One player hosts a room and friends can join to take turns choosing GIFs while everyone else submits the song they think fits the moment best.

Each round, players vote on which song matched the GIF the best. After a set number of rounds, the player with the most votes wins the game.

---

## How It Works

### 1. Host a Room
A player creates a game room that others can join using a room code.

### 2. Choose a GIF
The current round leader selects a GIF that will set the theme for the round.

### 3. Pick a Song
All other players choose a song they believe syncs best with the selected GIF.

### 4. Vote
Once submissions are in, players vote on the song that matched the GIF the best.

### 5. Score Points
The player with the most votes for that round earns a point.

### 6. Win the Game
After the configured number of rounds, the player with the highest score wins.

---

## Features

- Real-time multiplayer game rooms
- Turn-based GIF selection
- Song submission and voting system
- Score tracking across rounds
- Room-based gameplay with friends
